- ✅ **Latest Model** - Gemini 2.0 Flash with improved quality
- ✅ Perfect for development and testing

## Step-by-Step Guide

### 1. Go to Google AI Studio
- Visit: [https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)
- Or: [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

### 2. Sign In
- Click "Sign in" or "Get API Key"
- Sign in with your Google account (Gmail account works)

### 3. Create API Key
- Click "Create API Key" button
- Select "Create API key in new project" (or use existing project)
- Your API key will be generated immediately
- **Copy the key** - it starts with `AIza...`

### 4. Add to .env.local
- Open your `.env.local` file
- Find the line: `GEMINI_API_KEY=your-gemini-api-key-here`
- Replace `your-gemini-api-key-here` with your actual key
- Save the file

### 5. Restart Server
```bash
# Stop server (Ctrl+C)
npm run dev
```

That's it! The app will now use Gemini for all AI features.
